# 💸 AI Expense Tracker

An AI-powered expense tracking application that helps users manage their finances efficiently through intelligent insights, budget tracking, and interactive dashboards.

---

## 📌 Overview

AI Expense Tracker is a full-stack web application designed to simplify personal finance management. Users can securely track income and expenses, manage budgets, analyze spending habits, and receive AI-generated financial insights.

---

## ✨ Features

- 🔐 Secure User Authentication (JWT)
- 💰 Income & Expense Management
- 📊 Interactive Dashboard
- 📈 Spending Analytics
- 🗂️ Category-wise Expense Tracking
- 🎯 Budget Management
- 🤖 AI-Powered Financial Insights (Google Gemini API)
- 📱 Responsive User Interface

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/vipulreddy135/AI-Expense-Tracker.git
```

---

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file.

Example:

```env
PORT=5000

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=expense_tracker

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_api_key
```

Start the backend:

```bash
npm run dev
```

---

### Frontend Setup

```bash
cd frontend/web
npm install
npm run dev
```
